## SEO

SEO is the top priority for this site:

- Pages ship **server-rendered HTML** â€” the React app uses `client:load`, so crawlers receive real content (including the `<h1>`) without executing JavaScript.
- **Structured data** (`src/seo/structuredData.ts`) â€” LocalBusiness, TaxiService, FAQPage, Reviews + AggregateRating, vehicle listings, airport, breadcrumb.
- **Per-locale** titles, descriptions, and keywords; correct `canonical` + `hreflang` (`en`, `el`, `x-default`).
- **Sitemap** auto-generated by `@astrojs/sitemap` (`/sitemap-index.xml`) with i18n alternates; referenced from `public/robots.txt`.
- `trailingSlash: 'always'` keeps canonical URLs, the sitemap, and Netlify redirects consistent.

## Internationalization

Two locales: `en` (default, served at `/`) and `el` (served at `/el/`). Default locale is not prefixed (`prefixDefaultLocale: false`). Translation strings live in `src/i18n/en.ts` and `src/i18n/el.ts`.
